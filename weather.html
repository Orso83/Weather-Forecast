This application uses two different third party APIs to get the weather forecast based on the users zip code. The first API is from opendatasoft. The Opendatasoft API is used to get the users geographic location based on the entered zip code. The second API is from openweathermap. The Open Weather API returns weather data based on the users geographic location.
